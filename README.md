# ✨ VendorHub Client (Frontend)

Panel de administración para vendedores. Permite explorar catálogos, gestionar stock personal y realizar ventas en tiempo real con una interfaz moderna.

## 🎨 Características
* **Dashboard:** Visualización de métricas y ventas.
* **Caja (POS):** Registro rápido de ventas con validación de stock.
* **Inventario:** Gestión de productos propios y alertas de "Agotado".
* **Catálogo:** Importación de productos desde la marca matriz.

## 🚀 Tecnologías
* **React 18** + **Vite**
* **Tailwind CSS** para el diseño
* **Shadcn/ui** (Componentes de alta calidad)
* **Lucide React** para iconografía
* **Axios** para consumo de API
